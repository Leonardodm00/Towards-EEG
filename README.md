## Script description
- **automated_reconstruction.py**: Main script to perform neuron reconstruction directly from the google cloud storage.
- **Rec_Utility**: Compendium of functions  backing up '**automated_reconstruction.py**'.
- **final_get_comparments_h10.py**: Debug script for the function stored in '**Rec_Utility**'.
- **utility.py**: Other functions used for visualization, anisotropy-based alignment, minimum arbour length assessment, IDs check, etc...
